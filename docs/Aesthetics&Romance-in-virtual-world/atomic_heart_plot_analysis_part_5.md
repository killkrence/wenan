#右伶梦蝶——〈原子之心〉艺术与剧情解读释义

距离《原子之心》的发售已经过去有一段时间了，我相信大家也已经通过了各自的方式观看或体验完了整个游戏的剧情内容。在前两期的节目当中呢，由于那时候《原子之心》这款游戏还没有发售，使得我们很难窥其全貌。因此，前瞻性地讨论和观看了苏联美学中的构成主义以及至上主义，同时也简单地说明了他们与立体主义以及未来主义的某些潜在联系，也谈到了一些象征性的内容。而我们这一期的视频，将会把话题继续延续下去，同时这也证明了我们的推断并没有出错。通过发售前从预告片里所观察到的内容，这款游戏的美学，也真就停靠在我们的猜测范围之内。

我们这一期话题的开端是超现实主义，将会以艺术领域的内容作为起点，扩散到《原子之心》这款游戏的剧情。它的中心思想，或者更准确一点来说，应该是它的剧作，是在怎样的思想框架下被编织出来的。因此我们今天会不可避免地聊到一些哲学相关的内容，可能听起来会比较累，比较催眠。不过我想大家应该早就习惯了。

超现实主义是1924年起始于法国的一场艺术运动。那时候欧洲各国正值第一次世界大战之后的战争恢复阶段，欧洲人民的情感与思想也因为战争中人类的互相屠杀，因为那种残酷开始逐渐地收紧，开始变得向内。也因为这一原因，原本向外的艺术审美情操也渐渐地开始不能满足社会的精神需求了。这个“向内”和“向外”，我们在之前的节目里有解释过了，大家如果有兴趣的话，可以翻一翻我们之前的节目，好像是关于《艾尔登法环》的节目，对吧？人们呢，也开始探讨与向往更接近一种隐性的自然，也就是更接近于自己内心的艺术文化。而超现实主义所强调的，就是探求隐性的自然，探求另一个世界的艺术运动。大家应该都能听得出来，超现实主义多多少少都有点我们所熟悉的浪漫主义的味道，也就是反物欲引导下的物质，开始提倡精神能量。反社会规训下的虚伪，大声地号召人们要开始探求自我。反我们所生活的现实世界，转头宣扬另一个未知世界的伟大与真实。

只不过虽然说，呃，能在超现实主义与浪漫主义身上对照出许多的相似点，但其实这一艺术运动的源头并非只有浪漫主义。关于这一点，我相信大家都知道，主要是还是因为浪漫主义的影响面实在是太广，同时也非常的混乱，它自身就很混乱。以后我们有机会的话，应该会聊到浪漫主义的。实际上超现实主义的真正源头，或者说它所承载的精神母体，并不单一地指向浪漫主义，它更多的承载的是达达主义的灵魂。

达达主义是一个诞生在第一次世界大战期间的艺术运动。它发起并追问了一个在当时显得相当叛逆的问题，就是人类世界之所以变得如此的混乱和肮脏，到底是因为什么？就这个问题，达达主义也给出了自己的答案：正是因为人类自己。达达主义认为，人类经由启蒙运动之后，由于科学的后天介入，致使我们失去了信仰的庇护，变得相当功利与世俗。而后又经由隐藏在社会中的资本逻辑洗脑，导致我们开始不厌其烦地把欲望当作体面，以金钱来衡量价值。达达主义认为，资本主义用一种形式上趋于合理化的逻辑，掌握了知识的分配权，从而操控了社会的价值观，操控了社会的道德，同时也就操控了什么？操控了支撑一个国家得以行使的法律，以一种野蛮的姿态不断地影响了各个家庭，致使了每个人的悲剧，甚至国家的悲剧。这是国家与国家之间的相似悲剧，不断地重复地出现在人类的生活与历史舞台上。而达达主义的诞生，便是为了反抗这样的现象。也就像这个艺术运动的名字“达达”一样，如果我们去翻外文字典，就会发现这个词在不同国家，不同的语气当中拥有着不同的意思。而那些不同的意思就代表了达达主义的思想。简单来说就是“达达”这两个音节代表的是一种虚无，代表的是零，同时也代表了一切。这个“一切”的意义，包含了不被知识所左右的意义。

怎么理解呢？我们可以想象一下，大家可以想象一下，就比如“达达”可以表示高跟鞋所带来的提气与暧昧，对吧？“哒哒”也可以表示战争的声音，像开枪的哒哒哒哒哒哒。同时还可以代表一种否定，就像有些人在习惯否定别人的时候，喜欢那种咂嘴的声音，对吧？也正是达达主义的这种反抗精神，促使了这一运动下的艺术作品，都在强调或推翻或打乱欧洲当时已有的社会规则，其中包括了道德规则、知识规则以及人文规则。借此以强调现实世界的虚伪性质，主张非现实世界那种更接近于内心的，近似于感召一般的情景。就像这些我们已经看了有大半天的艺术作品一样。不难看出，达达主义更趋向于一种自我混乱的情感表达。

那么达达主义的运动最终成功了吗？那从艺术领域的角度来说，它成功了。至少它生下了它的孩子，也就是我们刚刚说到的超现实主义。而且相比它的母亲达达主义而言，作为孩子的超现实主义，有着更加坚定的方向与纲领。其中最具代表性与影响力的就要数安德烈·布勒东发表于1924年的《超现实主义第一宣言》。布勒东在这份宣言里定义了他的自由，阐释了社会规训，提到了弗洛伊德描绘了梦境。他说：“在文明的掩护下，以进步为理由，人们已经将所有对比于科学的，可以称之为迷信或幻想的东西，一律摒除于思想之外，并且禁绝了一切不合常规的探求真理的方式。”

我想说到这里，如果大家认真地或体验了，或观看了《原子之心》的剧情的话，应该都已经产生了许多的联想了。我们会发现，《原子之心》的剧情思想，其蓝图与背景好像就是建立在布勒东，也就是超现实主义运动的思想之上的游戏。剧情内的政治斗争恰恰不是剧情的导火索，而只是一种被蒙蔽在权力之上的虚妄，是一种思想上的枷锁。因此，如果我们剥离掉这层叫做虚妄的迷雾，便会发现剧情的主题，也就开始从传统俗套的阴谋论语气，走向了科学与自然之间的斗争，走向了自由与规则之间的斗争，走向了个人与理想之间的斗争。

那么，《原子之心》这款游戏有没有直观地体现出有关于超现实主义的内容呢？面对这个问题，如果我们单单只从狭义的艺术领域方向出发，以微观的视角去观察与分析的话，会发现这个世界并没有超现实主义的参与，更多的是我们在之前节目中提到过的构成主义与至上主义的形式表达。但如果我们一旦从宏观的视角，不只是从狭义的艺术角度去看待这部作品的话，就又会发现超现实主义几乎充斥在游戏的方方面面。其中最为典型的就要数我们的主角的梦境了。那这种梦境的设计就是非常非常典型的超现实主义的风格。

有一点值得我们注意的是，不要试图去解读类似于这样的在超现实主义之下构建的文学作品，或是其他艺术作品的某一区块。因为它们包括了游戏里的这一场景，虽然充满了象征性，但是却没有具体的意义。就像我们在上面说到的，超现实主义的母亲达达主义所强调的一样，意义是零是没有，但是这个零和没有又代表了一切。因此我们如果要进行阐释，它必定是包罗万象的。大家完全可以理解为，类似于这种超现实主义的表达，其象征性是需要我们放弃理性思考的。我们的知识在这里无效，我们的逻辑在这里无效，我们的后天经验在这里也同样无效。这就像我们之前在上一期里聊到的那一对芭蕾舞机器人所拥有的象征性下的模糊性一样，一切都是直觉在导轨。这同时也说明了，我们为什么在游戏的其中一个结局当中，在这个被游戏称为“林伯玉”的梦境里，所见到的是右伶而不是左伶的原因。我想这个大家都知道，因为左脑掌控的是逻辑与理性，而右脑才掌控幻觉、冲动和直觉。

在游戏中，我们的主角还被指派追寻一位叫做彼得·罗夫的科研人员。期间会去到一个叫做玛雅·普利谢茨卡娅的剧院。这座剧院的整体人文设计也反映出了超现实主义的风格，就像大家正在看到的这一桥段一样。然而，真正重要的恰恰不是在这座剧院的人文设计所反映出的超现实主义风格上，而是彼得罗夫这名角色，他简直就是超现实主义的化身。或者说，他像极了我们的现实历史中与超现实主义有关的一个人，这个人就是大名鼎鼎的达利·萨尔瓦多·达利。一位被艺术界誉为疯狂，但却自称乖巧可人的天才级艺术家，也是超现实主义阵营当中相当重要的一位艺术家。他的性格以及在艺术与思想上的早期遭遇，就和《原子之心》中的彼得罗夫十分相似。我们今天可能没有太多时间来介绍达利的光辉事迹了，因为我们即将要开启另一个复杂的话题，牵扯出一位在科学与哲学领域都同时散发着光辉的人，这个人就是雅克·拉康。

雅克·拉康既是著名的精神分析学家，同时也是著名的作家学者。他提出的理论在哲学领域也被广泛地讨论，其主要原因是他的理论建立起了除开哲学以外的第二条探索人类真理的通道。嘛，可以这么说，因此拉康的理论在艺术领域以及文学领域里是非常受到推崇的。那么我们为什么会突然提到拉康呢？达利和拉康有着怎样的联系呢？我们接下去要引述艺术家达利在自传中的一段话。达利写道：“我刚接到一位最杰出的年轻精神病医生的电话，他才在《米诺托》中读到我关于妄想狂活动的各种内在机制的论文。他向我表示祝贺，我对这样一个题目的正确科学认识令他吃惊，他想见见我，当面讨论一下这个问题。我们商定当晚在我位于巴黎高盖街的画室里会面。这临近的会面使我激动，整个下午我都在努力地起草一份我们要谈事情的大纲。实际上我满意我的各种观点会在一种科学的环境中加以考虑，唯一心想使我们初次交换意见这件事能正规的甚至有几分庄严地进行。”

没错，拉康对超现实主义运动的艺术以及艺术家们非常感兴趣，尤其对达利这个人最感兴趣。于是才有了达利自传中的这次会面。达利后来还补充道，在这一次见面结束以后，他们交谈甚欢，甚至得出了许多的共同的观点，其中有一个就是他们都反对构成主义。并且在这次会面之后，达利和拉康就约定了定期见面。其实我们都知道达利被骗了，拉康在当时虽然欣赏，虽然很欣赏达利的才华，但是这位精神分析医师的思想，在根源上是与达利不同，甚至可以说是完全相反的。他之所以与达利进行会面，一方面是觉得达利的才华与众不同，想要了解他自己理论下的对立面究竟如何；另一方面是他想要对达利进行精神分析，他觉得这会有助于丰富与完善自己的理论。现在大家对照游戏自称被骗的彼得罗夫，就应该能明白了。同时我们还能发现，游戏将拉康拆分成了两个人，或者说将拉康的早期理论与后期理论拆分成了两个人。一个就是谢切诺夫，另一个则是扎哈罗夫，也就是在游戏主导研发了有机聚合物的两位苏联的顶级科学家。谢切诺夫与拉康早年的思想论调相似，而扎哈罗夫，也就是手套查尔斯，则更接近于拉康后期的理论。

那么拉康的理论究竟是什么呢？如果要展开来说的话，我们这个节目的时间铁定是不够的，一定是不够的。我们今天在这里只片面地谈及一点，那就是达利与拉康对于人类当下与未来的不同观点。达利认为，知识与习惯共同维系着一种理智主体和理智化的生活。人们在日常生活中，依据现实体制来建构自己，建构生活关系，恰恰就在这种建构当中，人最彻底地失去了真实的自己。而拉康则认为，知识与习惯共同维系着一种理智主体和理智化的生活。人们在日常生活中，依据现实体制来建构自己，建构生活关系，而人在意图建构这种关系时，就已经彻底地失去了真实的自己了。简单来说，达利认为是社会蒙蔽了自我，我们要让自我通过艺术给予自我快乐，给予自我与自由。而拉康则直接宣布，也是当代第一个宣布自我已经死亡的人。

我们今天不会过分地展开拉康的思想与理论，那我们只会谈及这一点，也就是拉康认为“自我已经死亡”。学习哲学或者对哲学感兴趣的人，一定都知道黑格尔的“主奴辩证法”。拉康也有受到主奴辩证法的影响。我们现在如果围绕着拉康认为“自我已经死亡”的论点，如果片面地、不系统地通过主奴辩证法的方式来简单解释的话，就是在早期，他认为我们自出生以来就已经沦为了奴隶，而社会与生活迫使我们在主奴之间不断变换，就像游戏中谢切诺夫的思想一样。在后期他认为我们自出生以来就沦为了奴隶，是社会与生活迫使我们成为奴隶，始终都会是奴隶。打个比方来说就是，无论阶级高低，只要我们与他人形成了某种联系，这种主奴的关系就必定存在。无论生活环境，只要我们降临人世，就必定成为社会或生活的仆人。就像手套查尔斯，也就是扎哈罗夫，他骗了主角一样，那他是如何骗取主角的信任，甚至一步步地对他进行引导的，对吧？游戏给出的一个解释是，可以通过某种模块来控制主角的思想。这里暗示的就是科学对于人类的潜在规训。同时，如果我们以拉康的观点来整理完成，这种引导或规训，甚至是不需要通过科学与知识，仅仅需要一定的时间，就能透过语言影响主角的行为与判断。这种主奴关系的建构，就是一种心理力的建立。

就像我们下班回到家里，一身疲惫地吃完晚餐，那这时候我们的男友或女友提出说要出门散步的请求。于是我们便有了两种选择：答应或拒绝。如果答应，我们就变成了因为他人的要求而放弃遵从自身的想法。而如果拒绝，那对方也会面临我们刚刚说到的相似处境。这种联系不仅仅是人与人之间的，甚至猫狗也会影响着我们。而且不单单生命能影响我们，社会同样也会将这样的影响下落到我们身上，就像一个意识的集合一样影响着我们。并且自出生以来，这样的影响就会如影随形，直到我们的肉体的消亡。因此拉康的观点认为，人类不存在个人的自由意志。生活在任意一个体制或群体，甚至是自然中的人类，都不可能完全以遵从自我的方式去生活。自然也就不存在所谓的乌托邦或是天堂。如果有的话，真正的天堂，在拉康的嘴中，真正的天堂就是母亲的子宫。如果不回到母亲的子宫，那唯一能够让人轻松一刻的地方，只可能是超越现实存在的另一个世界——梦中的世界。

解释心理，还有一个更加直观的方式。现在大家看到的是一个由四条直线构成的正方形，它的内部没有任何东西。现在我在其中添加上一个圆形。因为这个圆形的出现，使得我们好像察觉到了另一些东西，好像更能感觉到这个正方形的中间点了。圆形就像一个坐标，指引了中间点的位置。这便是心理力的作用。我们之所以会觉得正方形的中间点变得更容易被察觉或确立，是因为我们人为地，我人为地添加了一个圆，是因为我在各位与这个正方形之间，建立了联系，并加以引导。当然也有少部分人会说：“啊，我完全察觉不到我们刚刚说到的那些东西啊，中间点什么的，感觉不到。”恰好这少部分人，就构成了游戏中的另一个论点，也就是在医院场景中讨论科学以牺牲少部分人生命为代价，拯救大部分人生命的行为是否合理。放到我们这里，或将游戏的话题延伸，就会变成大部分人是否需要去在乎、在意少部分人的言论，也就是所谓的言论自由。

（游戏内对话：这是什么地方？查尔斯，停尸间吗？没错，就是存放尸体，好拿来做研究的。你说什么？外面不到处是尸体吗？这停尸间的死人到底哪来的几十具尸体啊？大多还都是年轻人，都是志愿者，因为危险的人体实验死的。我知道有些实验不能用老鼠啊，猪啊什么的做，只能用活人，但这死的人也太多了吧。很遗憾，科学探索也不是一帆风顺的，时不时就会死几个人。建这个地下室放尸体，让人好好研究，就是为了以后少死点人。这样他们也不算是白死。去你妈的，时不时就不能拿死刑犯做实验吗？不过这也还是有问题，嗯，也不是什么实验都能拿社会渣子当小白鼠的，总有那么几个重要的秘密实验要求比较高，要找心理健康的志愿者。这些实验的结果有时候就比较惨烈。能比这还惨烈？少校同志，相信我，有些志愿者的下场比死还惨。不过有时候死了几个志愿者，就能让研究得到重大突破，挽救千万人的性命。褐色瘟疫的疫苗也是这么研制出来的。这一切必须到此为止了。）

只不过无论如何，拉康都不在乎我们的看法，因为他的理论是：无论我们是否在上面的那个正方形里，有没有感受到心理力都不重要，重要的恰恰是在这个小小实验的第一步，也就是我们认可了它是一个正方形。那么，一个自由且无畏的灵魂，一个完全自由的意识，为何会听从他人的吩咐去理解，一个抽象的图形就一定是正方形呢？拉康的意思其实很简单，启蒙运动并没有让人们觉醒，相反是让人们受到了更多的规训与教化，受到了后天基于科学知识的规则，一种自我奴化的过程。可能有人会问：“那如果不接受这些科学知识，社会就会停止发展，人类也会走向毁灭呀，对不对？人类文明就不会再发展了，甚至可能会倒退，可能会自我毁灭。”没错，这便是拉康理论最为要命的地方。正是因为我们必须接受这种规训与教化，才得以维持所谓的群体、社会乃至人类世界。所以自由才是一种虚妄，乌托邦与天堂才会是他口中的泡影，因为我们生来就成为了奴隶。

讲到这里，不知道大家还记不记得我们之前关于《极乐迪斯科》的节目当中，我们有聊到过的“我”与“自我”。我认为大家应该都还记得，对吧？而拉康的这个著名的理论，其实就是一种“伪自我说”，也就是我之前和大家提到过的，我们认为自己是身体的驾驶员，就像开高达一样。殊不知真正的高达驾驶员另有其人，我们只负责完成那个人交给我们的工作，只是一个奴隶而已。那如果我们失去了那位奴役我们的驾驶员，会发生什么呢？游戏就为我们构想了这样的一幅可怕的情景。

（游戏内声音：现在暂停偏振，这他妈是什么地方？是鬼屋吗？）

在这个基础之上，拉康还不止一次地声称，他最感兴趣的，恰恰就是那个自以为是主人的奴隶。这个口气听起来，是不是多少都有点像是扎哈罗夫，也就是我们可爱的手套查尔斯？

（游戏内对话：你想干什么，吉米特里？我想就到此为止吧，想让你们这些可悲的人类，明白自己根本没有未来可言。你们该靠边让路，准备迎接烟花的下一个阶段了。）

《原子之心》就是通过将拉康进行拆分，通过片面的示意拉康的理论，透过超现实主义的人文设计，表达了一种相当折磨、苦楚与矛盾的情绪。一方面，它传达了对于苏联那个祖国、那个曾经存在的祖国母亲的美好幻想，那份人人都能看到的伟大一面，那个曾经被理想所包裹的体制。另一方面，它也传达了祖国母亲在教化上的严厉与冷酷，那个人人都能体会到的大集体式的黑暗，那份为了达到理想就必须要以牺牲个人自由意志为代价的恐惧。游戏虽然以幽默的一面将其展现了出来，但也就是这样的展示方法，使得那份苦楚的焦灼更加戏谑，也更加令人印象深刻了。就像，就比如我们游戏中的这一段：

（游戏内对话：先锋队员，你以优异的成绩通过了达尔文测试。请问您长大之后想做什么？宇航员。多好的职业规划。衷心祝愿。少校，感谢您的关注。请问有什么可以帮到您的？）

但是也就像我们这期节目所聊到的内容一样，所谓的自由意志有可能只是一场谎言而已。自由意志只是不同社会规则下的不同游戏规则。没有人能够逃离规则，包括那些权力的争夺者，也只是参与其中的玩家而已。在这一点上，游戏还给出了一个很浪漫的，啊哈，很浪漫的一种幻想吧。它给出了一个叫做“林伯玉”的地方，告诉我们肉体的消亡并不代表着灵魂的湮灭，也许会有一个叫做“林伯玉”的地方等着我们。那个地方犹如超现实主义者所绘制的梦，那个地方就像拉康口中的梦境，那个地方也许是天堂。
