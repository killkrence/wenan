#记忆本是一出超现实主义剧场 《崩坏：星穹铁道》之黑天鹅
 与开拓者同行 第一期欢迎回来。在经历了几期特别节目之后，迎来了我们新一年里的啊第一期正式节目。在这里再次祝福各位新的一年里顺风顺水，万事如意。同时，就像频道里的老观众们都知道的那样，我们在这个频道里分享了许多人文领域、美学领域、艺术领域、心理学，甚至是神秘学领域的内容。而我们都围绕着它们整理过、阐释过，并将它们下落到了诸如达达主义、超现实主义、神秘主义、存在主义以及虚无主义的相关话题之上。而我们今天的这期节目，则会把这些内容融合起来，来看一看这些领域中的众多思想，在经过了人们精神世界的反复吐纳之后，当它最终变为人文领域中的一件娱乐制品之后，将会发生怎样的改变？它是否还会保留其本来的面貌？这期节目的主题是“记忆”，话题之下的对象，则是在B站这个平台上大家可能都耳熟能详的手机游戏制品——《崩坏：星穹铁道》。

在这里，先要为不太了解这个娱乐制品的观众们，做一个简单的背景介绍。《崩坏：星穹铁道》是一款手机游戏。从娱乐的角度来看，它与《王者荣耀》之类的电子游戏无异，都是令我们能够方便地通过手机或其他移动端设备来进行电子娱乐的文化产品。为了节省一些篇幅，我们接下来就会亲切地称《崩坏：星穹铁道》为“崩铁”。在大多数时候，崩铁这款游戏最叫人在意的就是它的人物设计了。而对剧作或人物设计有着一定了解的观众们，也肯定都知道，人物设计的关键在于剧本的好坏与否。而剧本的好坏与否，又关乎于它是否能够在不通过除开已有剧本以外的其他形式进行备注的前提下，也就是仅仅通过剧作本身，去精妙地映射出整个故事所在的世界观。譬如被改编的《指环王》与《哈利·波特》的舞台以及音乐剧，就是仅靠这种剧本反映世界观，甚至某些细微设定的绝佳范例。而在电影领域和动画领域中的《千年女优》，它所使用的超现实主义手法，仅凭镜头语言进行的偏意象的世界观表达方式，更可以用“经验”来形容。自然，我们今天的话题对象崩铁，也十分地注重剧作。主要是因为这个手机游戏里的角色，能够为创作方，也就是游戏公司本身带来经济效益。因为那些被设计而出的角色，有时候是需要花费真金白银去获得的。这也就造成了，角色成为了这款游戏的世界观架构中的非常重要的一环。

在前段时间，这款游戏更新了自己的全新版本。不了解的观众可以理解成，崩铁与“魔法石”已经发售了一段时间，而现在更新的这个全新版本，提供了一个可以延续之前剧情的全新故事包。继续用打趣的方式来说的话，就是它推出了崩铁与“密室”啊。希望大家不会介意。之所以用这么蹩脚的比喻方式，是我想尽量地缩短篇幅，以一个大家都能理解的方式，让大多数人都参与进来。在崩铁的这个全新版本当中，它也相当自然地推出了一系列的全新角色。第一个登场的人物叫黑天鹅，一名漂亮的女性角色。而我们今天的话题，就会围绕在这位女性的只言片语，并延伸到另一些角落去观察与总结游戏制品们喜好以怎样的方式来建构世界观，并以此进一步拼接出玩家们对除了剧情之外，同样也十分在意的游玩要素。在这之前，我们还需要为崩铁这个全新故事包，做下一段注解。也就是在这个全新的故事包当中，崩铁畅想般地建立了一个梦中的城市。人们可以通过下榻酒店的行为，参与到一场可以被称之为共同的梦境之中。而这座梦中的城市，这个被打造出的共同的梦境，它的名字叫做匹诺康尼。

听到这里，熟悉心理学领域的观众，肯定立刻就想起了许多学者——弗洛伊德、拉康、荣格等一众心理学领域的大家们，都曾为梦境做出过完全不同的解释。他们的理论互有反驳，得出的一些结论甚至完全相悖。这不由得叫我们今天的话题变得微妙起来。只不过大家也不用太过担心，因为我们今天所探讨的方向，将会以这些理论基石所共同认可的内容来作为支点，并且我也有十足的理由去自信地认为，频道里的许多老观众们，在知晓了这位漂亮的女性，也就是我们刚刚提到的黑天鹅这名角色，她其实与“记忆”二字息息相关之后，就肯定已经猜到了我们今天话题的大致方向了。

黑天鹅这名角色的一些只言片语，结合这场所谓的共同梦境，也就是匹诺康尼，以及心理学领域的诸多内容，很难不让人在意起一些事情。在剧作中，她曾向玩家抛出过这样的文字，大致是“在梦境中，人人都可以欺骗，但是记忆却不会骗人”。我也曾在像极了前瞻节目的新年特别节目里，引用了另一段小有名气的文字，对黑天鹅的这番发言作出了批注，就是“漂亮的女人最会骗人”。因为在事实上，记忆不仅仅会欺骗，且正是因为记忆的欺骗性质，才得以使黑天鹅能够站在一个所谓的记忆的角度，去对另一个个体的命运进行拆解，做出预言等。

在这里，我想先问大家一个问题：请问大伙儿还记得自己小时候的一些事情吗？OK，我想绝大多数人的回答都是正向的，也就是能够记得小时候的一些事情。在回答了这个问题之后，又出现了另一个问题：请问大家能够在脑海里完整地复原自己所记得的那些事情的来龙去脉，逼真地复刻出所谓记忆当中的那些场景、摆设、出场人物的细微表情、参演人员的电影台词，以及自己所持身份的具体脚本吗？可能大家会觉得我问的这些问题非常古怪、非常奇怪，就是为什么要将追寻自己记忆的某个场景，这种被称为“回忆”的行为，用戏剧化的代称指代出来？会将记忆中的我们自己称为“所持的身份”？将出现在记忆中的人物称为“出场人物”？将记忆中的事件、那些在脑海中的画面称作“脚本”？其实我们在这里的用意非常简单，只是想说明一件在心理学领域共同的认识，就是“记忆不可靠，记忆会撒谎”。如果用超现实主义的思想来勾勒那份诞生于记忆的谎言的话，听起来可能会比冷冰冰的心理学要显得浪漫得多，也就是我们标题上写着的：“当我们开始进行回忆行为的时候，一出超现实的剧场就在我们各自的脑海中拉开了帷幕。我们几乎无法阻拦那些所谓的、来自于过去的真实记忆，在被我们有意识地调动而出时，不会被大脑添油加醋，不会被增加细节、篡改结构，甚至演绎一个完全相反的故事。”

心理学领域中有一个共同的论调，就是我们会对记忆进行篡改。篡改的时效可能很短，只有数十分钟，但也可能很长，长达数十年。在这里还请各位不要误会，我们不是一期批判性质的节目。事实上，在心理学领域，我们个人对记忆所进行的篡改，在绝大多数时候都是一件值得高兴、相当乐观的事情。我们之所以会篡改记忆，实际上是我们各自的心理防线在发挥作用。它证明，至少我们的心理机制的运作状态，还处在一个可以调控或干预的水平线之内。也就是说，即使我们不幸罹患了十分严重的心理疾病，但凡只要还在这个水平线以内的状态下，也都是拥有痊愈的希望的。心理学领域的诸多学者们指出，我们人生中的第一次对于记忆的篡改，往往都发生在童年时期。我们可能因为环境因素而不可控地篡改自己的记忆，而这种篡改则通常都是在同理心被第一次激活的状态下进行的。而后学会以那种篡改来进行心理防御，保护我们面对突发事件时的心灵状态。在心理学领域中的诸多例子之中，童年的创伤性事件，应该是最能被大多数人接受，同时也是大多数心理学学者们都共同认可的事实。解释来说就是一名孩童，就比如在座的各位，我们各自的童年时期，当我们目睹了周遭环境突发的痛苦因素之后（当然这份痛苦有可能是鸟儿濒临死亡的挣扎，可能是蚂蚁昆虫们的扭曲，可能是同伴摔倒后的哭泣声，也可能是被欺负的受害者的控诉声）。在这些包含了痛苦因素的突发事件中，我们的同理心第一次被调动而出。痛苦就像是钉子一般的，从其他个体身上映射到了我们自身，令我们错以为发生了一些别的事情。就像我们经常能在托儿所、幼儿园等一些场所里所遇见的那种状况一样，明明是一个小孩子欺负了另一个小孩，致使了别人的哭泣，但他却也紧跟着一起哭了起来，并“恶人先告状”一般地向家长或老师报告，是他自己被欺负了。在心理学上，恰恰是因为他个体的同理心被激发，感受到了他人身上的恐惧和痛苦，导致了他在认知上的片刻混乱，进而篡改了自己的记忆，认为自己才是被欺凌、惨遭责难的那个真正的受害者。需要我们注意的是，这种状况会发生在我们大多数人身上，刚刚的那种案例并不是凤毛麟角的个例。相反，从来没有发生过这种情况的，才是异类。而在他们身上也极易出现各式心理疾病，其中最为著名的，也同时能够支持我们刚刚聊到的内容的，就是人格障碍以及自闭症。

聊到这里，肯定有观众表示不理解或不赞同。那至少肯定有人会笃定地认为自己一定不会丢失珍贵的记忆，更不会篡改自己的记忆。就此，我们现在需要了解一个概念，它的名字叫做“时间图像”。时间图像这一概念经常被运用于电影理论，那传达的是一份图像对我们个人心理世界所产生的影响，延伸的是一份图像所建构的意象世界。然而，当我们瞭望心理学时，时间图像的概念，连带着心理学对记忆和梦境的理论，便发生了奇妙的化学反应。举个简单的例子来说，就是当我们遭遇重大事件时，或者应该换个说法来解释，就是当我们遇见那些难忘的经历时，就比如某份非常快乐或非常痛苦的经历时，那份经历的感受，会形成一张犹如动图一般的短暂图像，印刻在我们的脑海里、心灵之中。这张短暂的动图是绝对的真实，是事实使我们能够铭记于心的一瞬间。当我们展开回忆行为时，通常都是从这张短暂的动图、这张时间图像作为意识的切入点的。就拿我们刚刚提到的围绕着孩童们的同理心案例来说，回想类似于那样的记忆切入点，多是从痛苦的。就像同伴们痛苦的瞬间、哭泣的表情、嚎啕大哭的场景开始的。而后我们才会进一步追想这张时间图像之前的内容，以及围绕着这张时间图像之后的内容。我想大家都听出来了，在这种被称为回忆的过程之中，令我们产生记忆不会出现谬误的错觉的，正是那张时间图像。因为它是绝对真实的，是一个拥有健全大脑与积极心灵的个体所不会轻易篡改的内容。只是一旦当我们开始追寻围绕着这张时间图像的之前或之后的内容时，事情往往都不会按照那种我们所希望的追溯事实的方向发展。我们的大脑会根据我们各自先天的或后天的人格特质，对记忆进行轻微、甚至是严重的篡改，也就是加入我们被意象世界影响之后的想象性质的内容。这便造就了类似于这样的状况，那便是当我们与他人分享回忆时，那张时间图像是大家都认可的内容。共同经历过一起重大事件、难忘事件的人，都会认可他人各自不同的时间图像，其内容的真实性与准确性。但当所有人将整个事件的前因后果、来龙去脉复述出来之后，却是大相径庭，甚至是完全相反的。我们大可以这样去总结，我们的大脑连携着我们的心灵，对某起事件产生了反应，并由意识按下了犹如照相机一般的快门，记录下了一张时间图像。同时也因为这张图像的重要性，它被大脑保管在了最为安全与缜密的场所里，方便我们及时调取这张拥有着绝对真实性的时间图像。

而当这一理论下落到我们今天的话题——电子游戏当中，时间图像就从意识的海洋与无序的心灵涌动，演变成了某种可视化的事物。近期的一些游戏称它为“烙痕”啊，有一些老游戏叫它“里庄”，而在崩铁这里，则被称为“光锥”。自然，如果大家有兴趣的话，去搜索“光锥”二字，会赫然发现它是物理学领域的一个专业名词，会看见爱因斯坦的相片出现在自己搜索条目的正下方。然而，就像我们在这期节目的主要话题一样，如果仅从心理学的角度来考量“光锥”二字，它同时也像极了一份来自于心理学领域的暗喻或意象：一起重大事件被烙印在了我们的心灵之中，而这一切的始作俑者，就是一把被称为“时光”的锥子。

现在我们知晓了记忆的谎言，知晓了那份来自于由我们自己编织的谎言，知晓了记忆的欺骗性，以及它在哪些地方会欺骗我们。那接下去，我们不妨来看看上面提到的黑天鹅这名角色是如何利用这种记忆的谎言来进行预言的。在这里我们就又不得不提及另一个理论，它有着一个非常容易叫人理解的名字——“控制论”。起初，这一理论，也就是控制论，仅被用作科学研究，而后发展并延伸到了诸多其他领域之中。在今天，我们要谈及的是控制论在20世纪初的新行为心理学中的含义。控制论的理论核心的支点是“信息”二字。在新行为心理学中，作为支点的信息二字，其核心方向被规制在了如何通过信息的方式，较一个个体的运行轨迹朝着可能性空间较小的方向发展。什么是可能性空间呢？我们这里举个例子，就比如我们现在啊，身上身无分文，并且饥饿难耐。但让人高兴的是，有一位天使愿意资助我们饱餐一顿。在这个前提之下，假设一条热闹的街道上有着三家餐馆，前两家餐馆的最低消费至少是十元，而第三家餐馆的最低消费则是五元。此时，如果天使想要控制我们的运行轨迹，也就是控制我们的行为，只需要简单地做出给我们十块钱还是五块钱的选择，就大功告成了。如果给我们十元，那我们之后的运行轨迹的可能性空间就会较大，因为我们至少能在三家餐馆之间做出一个选择。但如若天使只给我们五元钱，那么也就意味着我们接下去的运行轨迹的可能性空间，就会变得非常低，甚至只有唯一的选择了。20世纪新行为心理学的学者们认为，更有行为主义的有识之士认定，信息二字就是我们刚刚那一例子之中的、愿意资助我们饱餐一顿的天使。它几乎左右了我们在日常生活中的可能性空间的大小。在某种程度上，在某些情况中，它甚至能够通过或温柔或剧烈的方式，将我们各自的可能性空间、将我们的行为方向降低到一或一以下，也就是没有任何选择的权利，或直接放弃去选择的想法。注意，我们这里说的信息不是短信、简讯，而是我们通过眼睛、耳朵、鼻子或任何方式能够感受、理解并接受的信息。我想，对于科技越发发达的当下，有许多人已经有了这种可能性空间被信息影响，从而不断减少的感受了。最近我收到了一份小礼物，是一块苹果的电子手表。具体内容因为篇幅关系，我不会在这里展开去细说。但是这块小小电子手表所搭载与建立的、仅围绕着我们个人的集中信息机制，就能轻易地摆弄一个人的向左或向右。

我们先回到今天的话题上来。现在我们不妨畅想一下，畅想各位拥有了一种超能力啊。这一能力能够帮助各位减少其他个体的可能性空间，并朝着自己想要那个个体运行的轨迹方向前行，从而做下一段像是预言一般、实则是心理学意义上的行为暗示或行为控制的行为。这个超能力不是读心术，也不是复刻、重现或知悉对方的全部记忆，而是能够像打扑克一样地将他人内心中的时间图像，犹如卡牌一般地分离出来。想象一下，如果现在各位拥有了这种超能力，那么各位可以试想一下，能从身边的单个个体身上抽出多少张光锥、烙痕、履历（也就是时间图像）呢？大家在畅想的过程中，也一定能够意识到，这种知晓他人生命中绝对真实一刻的价值，要相比获悉他人的全部记忆（包括胡编乱造的幻想与不切实际的被篡改的记忆），来得要有分量的多。那么如果是各位，大家会想到怎样的方法去运用这种超能力，去控制一个人的行为，从而对他人做下所谓的预言呢？答案其实很简单。与其说在这种超能力的加持下，我们进行的是一场预言或占卜，倒不如说在掌握了他人巨大的秘密之后，进行了一场人为的推理，一场建立在现行社会规则之下的逻辑推理，一场由谎言连接在真实与真实之间的推理。只需要将这些照片一般的时间图像串联起来，就像侦探们的线索墙一样，我们就能合理地猜测到照片主人的下一步行动，其最小的那个可能性空间在哪儿了。而连接图像与图像之间的谎言，也恰恰成为了佐证时间图像其真实的证据。更可以说，在我们大脑篡改的那些谎言的帮助下，真正的时间图像才得以被推理出来，摆放出来，才得以犹如卡牌一样地从心灵中被抽出。打趣来说，如果将这个能力交给福尔摩斯的话，我想他能借助自己的演绎法，将这种预言做到极致，也许能够精确到一个个体接下去的每一个小时。就像在崩铁，黑天鹅、花火与桑博进行的那场侦探游戏一样，我们能发现，实际上心理学领域的诸多内容都被运用、下落在了《崩坏：星穹铁道》这款游戏之中。特别是在它最近推出的这个故事包里，在匹诺康尼中，大量心理学的内容都随着另外两个字，也就是“梦境”二字浮现出来。可惜的是，到目前为止，除开黑天鹅的只言片语以及记忆相关内容的描写之外，崩铁这部作品对于梦境的刻画更趋于娱乐化，它更像是在一个虚拟世界的基础上，去搭建另一个虚拟世界，营造的是一种氛围之下的感受，而不是氛围本身。

同时，我想各位从我们今天的节目也都发现了，许多电子游戏其实都引用了人文领域中的，特别是心理学领域中的诸多内容，作为其游戏行为的依托。就比如“光锥”这种对于人物战斗力扩充的物品，比如“烙痕”啊，那种提升人物战斗力必须携带的物品一样。大可以说，正是人文内容搭建出了一款电子游戏玩法的拼图碎片。一款小小的电子游戏，当它被打上了娱乐标签之后，很可能会出现那种叫人认为它微不足道的情绪。但它其实也是由诞生自人们的文化精神，绵延与反复吐纳之后的结果。是数代人文精神的总结与沉淀。大可以说至少在我们频道这里，电影、电子游戏、绘画或雕塑，它们在一定程度上，是并列在人文领域之中的，没有地位高低之分。特别是当时间来到了我们这个网络时代的当下，当时间来到前人预言的“虚拟走进现实，逐渐的发展成现实走进虚拟”的这个当下，一切就都变得明晰起来了。

OK，我们今天的这期节目啊，浅显地聊到了有关于记忆以及新行为心理学的相关内容。话题指向的则是热门的手机游戏《崩坏：星穹铁道》。如果恰好你也是这款游戏的玩家，我猜你也会对黑天鹅这名角色有了全新的了解。至少我们都清晰了，记忆有时候会如同命运一般地捉弄我们每一个人。谁能准确地预言人生的下一个街角转口，有着怎样的期许在等待着我们呢？也许那张小小的时间图像，黑天鹅手中的那张小小的卡牌，能够回答一部分的疑惑。只不过这些内容还牵扯到心理学领域中最为晦涩、有些学者更将其称之为“灰色的地带”，也就是暗示与催眠。它同时也连接了一部分玄学和神秘学的内容。这些话题我们不妨留到之后有机会再来聊了。那话说回来，崩铁里有一名角色好像就擅长催眠来着。他好像能够通过催眠缓解一种叫做“摩音症”（Morgellons disease）的病症。那么摩音症究竟是心理病症还是生理病症呢？或者应该这样问：催眠的治疗范围是否能够涵盖生理病症呢？催眠是否是科学的呢？这是一个有趣的问题。OK，我们今天的节目就先到这里。感谢各位的收听与收看，我们之后再见，拜拜。
