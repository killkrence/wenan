
#死亡的“价值”

大家好，我们又见面了。先感谢，在节目开始之前，先感谢各位的热情和支持，那么多的点赞和收藏，真的是给了我极大的鼓舞，感谢大家。这是我们与《极乐迪斯科》有关的第四期节目了，非常高兴大家能平和地讨论，耐心地坚持到咱们这第四期节目。说来惭愧，主要是因为我们前三期节目其实并没有深入地去聊一些东西，但是呢，它们又是必不可少的铺垫。如果用阶段来说的话，那么我们这个就算是第一阶段已经完全结束了，那进入了第二阶段了。我们的主题会从之前的话题，做出一个延伸和转变。在开始之前，我们最好为之前的那一个阶段做一个总结。这个总结非常地有必要。

我们之前花费了大概两期的视频，50分钟的内容，一起整理了一种群体的心理状态，以此去更好地接受一种现象，那就是我们个体之间是存在着差异的。而拥有着大致相同模样的群体与群体之间，也存在着差异。就比如我们之前对比出生自不同年代与文化环境的人与人之间的差异一样。我也聊到了不同时代的社会文化形成以及社会规训是影响、诱导、致使这种差异出现的原因之一。也以此，我们将话题引申向了文学领域的意识流。而在今天，我们要开始讨论的话题，将会是意识流文学喜爱调用的意向，而要理解这种意向，就脱不开我们之前两期节目里所讨论到的那种个体与个体之间、群体与群体之间的差异。

我们今天的话题是直觉与理性。在这个话题之下，还隐藏着另一个较为深沉的、沉重的话题，就是我们在上一期结尾说到的死亡。对死亡话题感兴趣的人，肯定都听过耶鲁大学的一个公开课，我分了三次才将它看完。我想说的是，我们今天不会采用注重逻辑的分析哲学的方式来剖析死亡。我们的这个话题是围绕着文学的审美展开的。当然，这也并不代表着它不会有着哲学的参与。如果大家能接受，我是更愿意从文学与美学角度和大家说一说死亡这一话题的。所以我想说的是，请大家放心，我们聊到的内容不会那么的，怎么说呢，就是像是分析哲学那样的烧脑。相反可能需要另一些补充，就比如业余时间里的大量阅读，以及在面对不同文化时都抱有浓厚的兴趣，去了解与探究的精神。

OK，我们都知道死亡其实是一个很大的话题。围绕着死亡的议题每年都在增多。而在艺术领域或是文学领域当中，死亡呢，又伴随着另外的这样的一个问题，就是死亡拥有价值吗？估计很多小伙伴、观众们在我发出这样的疑问之后，都开始寻找答案了。大家先等一等。我们在探讨死亡是否拥有价值的时候，要先探讨另外的一个问题，就是“价值是什么？”这个问题里的价值究竟指代的是什么价值？我们上一期就有聊到过，不同时代文化、不同社会规训成长的人们所看待事物的眼光都会不同，这同时也导致了他们对价值问题会给出不同的答案。而在价值概念下，直觉与理性就起到了很大的作用。

在我们当下的社会文化环境当中，人们在欣赏文学作品、研究文学作品时，或自足娱乐时、观赏艺术时，都更加容易接受拥有意义的内容。那些符号性质的标签、性质的意义，就像在我们当下许多人在欣赏，比如说荷马的《伊利亚特》的时候，都非常在意书中出现的符号，就比如水，就比如圆形，就比如方形。随之而来的就是圆形以及方形与水之间的关系。当然我们这里只是随便举个例子而已，大家不用那么地认真。其实就是试图用具象的方式去还原一个抽象的情景。依据也当然就是雅典人热爱几何，希腊人迷恋水。那许多游戏也是一样，有很多人在聊到《黑暗之魂》或是《艾尔登法环》那种可能缺少缺乏一些剧作与叙事的游戏的时候，就喜爱用那种偏向细节的视角去将一些细节以意义化，形成一种类似客观的理解。这其实就是一种偏向于理性的观察事物、总结事物与感受事物的方法。在当下呢，这种异化是相当受到追捧的。那大家在 B 站上面随便看一看，搜一搜《艾尔登法环》的一些解读类的视频，就能知道这种受到追捧的程度了。可以说我们现在的文化环境更流行以理性的视角去看待事物，排序、整理、细分、诠释，然后再加以总结，几乎都是这样的一种模式。

相比之下，直觉就与理性完全不同。我再啰嗦一句，就是我们在上一期已经聊到过了，不同时代文化社会规范下的群体都是有区别的。如果大家已经理解了我们之前聊过的内容，那么也就能够很自然地通过了解一个时期的文化传统、社会规则，去描绘一个时代群体的大致轮廓了。就比如欧洲人看待火这一事物，与我们看待火这一事物，在意向上就有可能完全不同的。就比如现在的欧洲人在看待火这一事物时，与中世纪时期的欧洲人看待的火，就有可能是完全不同的。在中世纪那个偏向信仰的社会文化环境当中，在那个由信仰构建的社会规则下，人们相信地狱是真实存在的。也因此，火所带来的意象，或是其背后的那种含义，和我们现在肯定是有着非常大的区别。中世纪人所理解的信仰的火的意象，就是偏向直觉的。那说到这里，大家可能会出现一些误会，可能会不同意这样的说法，这很正常，我们先继续往下，误会就会慢慢地被解开。就此我们就会发现另外一件事情，就是价值是可以被后天、可以后天被人赋予的。就比如说宗教信仰，还是上面的那个例子，中世纪的人会认为地狱的烈焰是理性的，但是在我们当下，我们会认为是直觉的。相对的，我们会认为手机这样的事物是理性的，但是如果穿越回到中世纪，让那一时期的人们去理解手机，他们一定会认为手机是直觉的。不知道大家有没有看过《好兆头》啊，那部小说就很好地描绘了这种理性与直觉相互交替的、犹如螺旋阶梯一般的奇妙情景。

好的，我想我们上面所造成的误会也已经解开了。以此出发，我们又会发现文学作品其实也保留了这样的特质。也因为文学作品保留了这样的特质，而导致了意识流这一文学形式被认为是相当惊人的，是现代文学领域当中非常重要的一个区块。其实很多学习历史、学习文学或是哲学相关专业的各位，应该都深有体会，就是当我们在阅读中世纪时期的某某主教、某某领主去观察世界、总结自然的文章，或者细读他们的来往书信的时候，都会产生出一种偏差感。这种偏差就是两股来自于不同文化、社会、规则、环境的意象相互交融的那个过程。我们通常会以主观的视角去阅读那些文字，我们会鬼使神差地用自己当下的文化环境和社会规则规训下的经验，去审视那一时期的文字。在这一过程当中，我们的脑海里就已经出现了意识流文学致力于进行表达与描绘的情景了。换句话来说就是，我们通常都会以一种直觉的思考方式，去理解与分析脱离了我们当下世界的文化、环境与社会规则的事物。

于是在让我们回到价值的问题上来，那死亡是否拥有价值？有一种说法，死亡的最大价值就是它的未知，以及伴随着那份未知之下的恐惧，借此以让生命对其保持敬畏，借此鼓励生命生存下去。经过上面的讨论，我们能得出，这在我们当下是一种趋于直觉的观点。与之相对的，另一种说法认为，死亡代表的是虚无，无论任何生命在消亡之后都会归于沉寂，生命的价值也就此结束了。呃，这在我们当下是一种偏向于理性的观点。只是我们会发现，这两种说法似乎都有着一些矛盾的地方。直觉认为死亡的价值来源于恐惧，来源于人为赋值的意义，也就是在民俗、宗教驱使下的动力。理性理解的死亡的价值来源于生命自身，来源于消亡与否、存在与否。二者都没有提到死亡本身。那这样一来，我们就发现了一件有趣的事情，那就是人类还没有参透理解的事物，是无法对其进行真正的赋值的。而那些无法被赋值的事物就是永恒的，所以说死亡就是永恒的。

在这里我们还得理清另外一件事情，就是赋值这一行为，在文学意向下，甚至是哲学层面上，赋值都拥有着两层不同的含义。概括来说依旧是我们今天的主题，就是直觉与理性。举个简单的例子，也是老生常谈的例子了，两块石头，一块是石头，另一块是钻石。它们被摆放到我们面前，而我们被询问这两块石头哪一块价值更高，究竟是普通的石头呢，还是钻石呢？一些人会说钻石的价值更高，因为他之前2克拉就多少多少万对吧。而另一些人会说石头的价值更高，因为他可能注意到了石头上画着的爱心，他会认为这块普通的石头被人为地赋予了生活的意义，而生活的意义是无价的。通过这个例子，我们会发现赋值似乎有着一种规律，就是我们总是先对事物进行直觉层面的赋值，它的价值体现在生活之中。而社会则会对已经被直觉赋值了的事物，再进行理性的赋值。就像钻石一样，那句话怎么说来着？“钻石恒久远”，对吧，“一颗永流传”。这个广告，就是在表明了钻石被直觉赋予了价值之后，社会再以理性的方式对其再次进行赋值。就此我们会，呃，我们就能发现与总结出一些无法被赋值的事物。就比如说地狱，我到现在还没听说过去地狱观光要花多少钱，对不对？我也不知道第8号当铺到底在哪儿，解忧杂货铺到底存不存在，人头气球是多少钱一个，球状闪电大家有看过它的标价吗？对此我们也会发现，死亡可以被直觉赋值，但是无法被理性赋值的。因此我们再整理一下刚刚的那些论点，它就变成了在当下社会无法对其赋值的事物就是永恒的，死亡就是永恒的。

在《极乐迪斯科》这部作品下，我们就会联想到一些可怕的内容，就比如我们有没有被社会赋值？人有没有被赋值呢？大家不要认为我跑题，这和死亡这一话题其实息息相关。但很多人会说我们其实已经被社会赋值了啊，工资多少，工作面貌如何，职称高低、开的车是什么牌子，包包的限量与否，对吧，都影射了一个人的价值。确实如此，但是如果我们仔细地学会分析的话，就会发现，这其实都还只停留在直觉层面的赋值。我们能分析出一个人会不会挣钱，他会不会挣钱，但是我们始终无法笃定一个不会挣钱的人就是没有价值的。因为我们深知有一个叫做自我价值的东西。现在至少到目前为止，社会为我们打造的价值观念还没有压垮自我价值的重要性。即使社会中的资本一直在致力于打造出一种社会价值、金钱价值即是自我价值的观念，但也依旧没有完成一个过渡，原因就在于它无法将死亡赋值。文学世界里赛博朋克的世界，大家应该都知道，为什么在那个世界人可以被完全赋值的原因，也是因为死亡已经可以被赋值了。心脏不行了就换个机器心脏，肝脏不行了就换个机器肝脏，一切都可以用金钱确定其价值的人造物对其进行取代。而许多赛博朋克类的文学作品，就会又开始探讨，在那个死亡已经可能被赋值的世界里，意识会不会发生改变，也就是如果我们全身上下都被机器所代替了，那我到底还是不是我？以此能看出，赛博朋克作为现代文学领域的作品，依旧是在用直觉的方式处理理性的内容，讨论的依旧是、探讨的依旧是死亡观与意识。

我们先回到《极乐迪斯科》。那我们会不会有一天被社会真正的以理性的层面赋值呢？答案是有可能的。这就牵扯到了另外的一个话题，同时也是游戏前半段出现很频繁的一个话题，他就是种族主义。没错，种族主义就喜爱对我们进行真正的赋值，并且它还有着很多的类型。我们今天要聊到不是我们印象中法西斯所行使的种族主义，也不是肤色上的种族主义，而是我们当下社会中出现的一个现象，许多学者已经将它总结成了“行为种族主义”。我在 B 站收到了很多私信，许多人都表示看不太懂那个种族主义的大个子究竟在说什么。其实我们去理解这个大个子的所谓种族理论时，不能抱有传统的那种种族理念，就比如我们刚刚提到的那个法西斯式的理念，去理解它的理论。其实更倾向于我们刚刚说到的行为种族主义。那么什么是行为种族主义呢？其实解释起来相当地简单，就是将我们日常生活的行为、爱好、取向、认知进行区分，再去以此划分等级，甚至是内在价值的一种行为。这种行为种族主义，在我们当下这个信息发展迅速、高度网络化的生活环境当中，体现在了一些细微、不具体、分散的文化群体里。就比如有一种行为叫做“查成分”。这个所谓的“查成分”，其实就是利用一个个体碎片化的网络行为，图形化他人的生活行为，以一种不严谨的方式去区分和归类他人的爱好与特征，并借此去判定拥有着不同图形的人群在社会文化中所处的位置。首先，这种区分与种族主义是一样的，是相当盲目的。其次，这种区分的行为所导致的后果，和种族主义导致的人种与人种之间的对立一样，它会造成文化与文化之间的对立，进而造成阶级与阶级之间的对立。就像网络上嘲笑，呵呵，嘲笑什么来着，嘲笑玩《原神》的人都是 OP 一样是吧，借此去区分他人的位置，区分他人认知世界的能力。这其实就是一种相当盲目与危险的行为。可能有人会说有些东西事出有因的，是他们怎么样怎么样如何如何，这我都知道，我也都明白。只是有更重要的一点，大家也必须要明白，如果我们继续这样依靠区分他人的行为去概括他人的认知，映照他人的位置，那必定是所谓的资本乐于见到的事情。所以如果我们从这个角度去理解那个大个子所说的内容的话，我们甚至都不用去区分他口中说的那些名词，都能够明白他话里的含义。

在行为种族主义里，还牵扯着许多不同的文化运动，就比如近两年来话题度非常夸张的女性主义。在这里，大家可能不知道我为什么要提到女性主义，毕竟这跟我们上一次聊到《战神5》的希芙与芙蕾雅不太一样，那因为那个是有关于女性的，我们这里好像与女性无关。那这里我还是说一下，其实我们并没有跑题。那说到女性主义的原因，是因为这其中的一个话题叫做“凝视”。呵呵，女性主义强调的，而有许多女性主义强调的是男性凝视。那许多男同胞可能都无法理解这种凝视。我这里有一个小小的故事和大家分享。在这里我先强调一点，我尊重各式文化群体，尊重各式取向，也尊重有着各式审美爱好的人。我在这里所举出的例子只是为了帮助大家更好地理解什么是凝视。

我以前读书的时候穷学生嘛，对吧，怎么说呢，就算是一个穷学生也可能会有着一个比较花钱的爱好。哼哼，我那个时候是喜欢收集 CD，就是那种打口碟、走私碟。那如果大家不知道这个的话，大家上网一搜就知道了。总之，我和另外的一个与我一起长大的朋友，决定去一家连锁店打工挣零花钱。那家店开在闹市区，是一家专门销售潮流类手表的门店。那店长是一个相当热情，也是相当温柔的人吧，可以这么说吧。反正在那边打工的日子是很愉快的，因为我们我和我朋友都是刚刚开始接触工作的糊涂蛋，所以经常会和一些比较麻烦的顾客发生所谓的一些矛盾。我们那个店长都能帮我们解决掉，而且他会在顾客面前极力地维护我们。那重点来了啊，这些都不是重点，重点在下面。有一天店长邀请我们出去玩，我们一口就答应了。我记忆犹新，那是一个晚上，下着小雨。店长带我们去了一家酒吧。那那是我和我朋友第一次去酒吧。我刚进酒吧的时候就有一些不自在，因为我觉得我不太适应那边的环境，不是那种特别善于社交的人，善于交流的人。于是我就没说话，一直没说话，就坐在一个角落的沙发里，安静地观察着酒吧的环境。慢慢慢慢地，也不知道是过了5分钟、10分钟还是20分钟、半个小时，我开始觉得有那么一点点的古怪。因为似乎整个酒吧里的人都是男性，并且开始有目光稀稀落落地停留在我和我朋友身上。我也开始发现这些男性在打扮上可以说可以用“细致入微”来形容吧。干净的装扮，平整的指甲，对吧，还有相比我们那会儿简直就是压倒性的那种衣品穿着。很快，那些人的言谈举止，也让我意识到了另外的一些东西。因为那会儿还挺年轻的，我就开始有点慌了。一慌张就想往厕所跑，估计很多人都和我一样。然后才意识到我根本就不应该去厕所的，呵呵。

我想大家应该都猜出来了，那个店长带我们去的是一家 Gay 吧。这是一件真实发生的事情。我还记得我当时慌张地想去洗手间冷静一下，然后又怕别人觉得我是神经病，于是伪装尿尿，结果紧张到尿不出来的那种尴尬。期间我还不停地感觉有人在盯着我，最后只得落荒而逃。其实大多数女性主义所说到的凝视，就是这样的一种被凝视的感受。在这个凝视的感受之上的是一种意象，一种将一个或性别、生命、或物化或贬低的情节。有时候都会觉得有点可怕，因为现在这种情况其实还是挺麻烦的。可能很多男同胞不是很能理解。我不是经常会没事翻翻抖音嘛，我个人也挺喜欢一个中医的，那我觉得那个中医讲话特别有趣也很有意思。结果最近在他那听到偏向于，就偏向于这种行为种族主义的性别歧视。他将中国哲学阴阳的理论划分到了，划分到了，区分到了男性生来就应该做什么，女性生来就应该做什么。我们这个就不继续说了。在这里要多说明一点，就是我在这里提到的意向是凝视前提之下的意向，与我们刚刚所说到的那个例子其实没有太大的关联。那个例子之所以说出来给大家听，是让大家能够更好地明白有时候身为女性的感受，这是男性极少能体验到的。也是为了向大家说明《极乐迪斯科》在提及种族主义相关话题时为什么要使用测颅这样的一名角色。

测颅这名角色相当地高大，这是游戏告诉我们的。他是一个种族主义者，这也是游戏直接向我们明示的。大家可以想象一下，一个大高个，2米多，一身肌肉的，以一种不可一世的姿态俯视你的时候，你是一种怎样的感受？如果这个大个子还满嘴种族主义理论的话，你又会是一种怎样的感受？没错，你会感受到一种凝视。你会感觉对方并没有将你当作一个无法被真正意义上赋值的人，而是一个可以被真正赋值，甚至是可以被随意归类的物品。很快你会发现，这种凝视并不单单只存在于这位种族主义的混蛋身上，它还充斥在游戏的其他地方。对比我们的生活来看也是一样的，因为社会规则直觉性地将我们赋值，导致了我们会有一种被文化、社会群体，甚至是被本应该热爱的生活所凝视的直觉。就像《极乐迪斯科》所出现的那些政治话题与论点一样。而唯一能让我们保持清醒的，又是那些无法被社会赋值的事物的凝视。就比如信仰，我们这里说的信仰不单单指宗教，就比如我们开头就在说的死亡，死亡的凝视。社会与文化的各种凝视，一直都是现代文学的探讨范畴。而死亡的凝视就像这幅画一样，现在大家看到的这幅画是艺术家爱德华·蒙克的著名绘画作品《呐喊》。我想大家都知道，它所传达的意向就是一种被死亡凝视的意向。绘画强烈地表达着一种痛苦，一种被凝视之下的焦虑与忧愁。于是才有了这样的呐喊。但其实它也警醒着我们，死亡在带来那唯一结果之前，我们究竟应该以怎样的姿态去面对它。
